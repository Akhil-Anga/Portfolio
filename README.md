## Local Development

Clone repository:

```bash
git clone https://github.com/Akhil-Anga/Portfolio.git
cd Portfolio
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build production version:

```bash
npm run build
```

Deploy:

```bash
npm run deploy
```
